用Markdown语法编写blog

  • 一、什么是Markdown语法

    Markdown 是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。

    用易于编写的纯文本格式进行编写,然后将其转化为结构有效的HTML或者XHTML

    简单的来说就是它可以帮你编写出自带特效的文字,比如你现在所看到的就是Markdown语法编写的

  • 二、熟悉Markdown常用语法

    • 拥有Markdown编辑器

    1.typora

    2.vs code 需要安装插件 Markdown all in one 实时预览快捷键ctrl+shift+v

    3.vim

    • 开始熟悉语法

  • 标题

    语法:#空格 + 你的文字

    标题用 # 这个符号

    用1到5个#,可以理解为HTML中的h1到h5

    标题1

    标题2

    标题3

    标题4

    标题五

  • 菜单

    语法: *或-号空格 + 你的文字

    菜单用 -或*号都可以

    通常用于列出一些必须安装的东西或其他的清单

    • 西红柿
  • 哈密瓜
  • 大白菜

  • 引用

    语法: >空格 + 你的文字

    通常可以引用里面突出你的内容

    窗前明月光
    低头思故乡

  • 粗体或斜体

    粗体:

    语法: **你的文字**

    斜体:

    语法:*你的文字*

    作用是强调或者突出一个词

    我是粗体
    我是斜体

  • 高亮强调

    语法:`你的文字`

    这个符号是在ESCj键下面

    作用也是突出强调你的文字

    注意:
    卖大白菜的来了,赶紧买白菜

  • 代码高亮

    语法:

    ```语法类型

    你的代码

    ```

    ​

  • 链接

    语法:[描述](链接)

    博客

    可以点击这段文字后跳转到这个链接

  • 插入图片

    语法: ![]()

    这个 …需要用到图床…网上有免费的sm.sm

    ​

  • 插入分割线

    语法:***

    一般用于一大段就可以用一下分割线,会显得整洁一些

  • 9.插入视频

    额 这个可以用嵌入代码的方式或者用a标签
